
current: log(hue($cw.wiki.getTiddlerText('REPL-MOTD-MSG'), 40))

Displays the text of tiddler 'REPL MOTD' from the code wiki.
log is shorthand for console.log()
hue colorizes text using the 256 terminal color codes
$cw is the tiddlywiki instance connected to the code wiki
  (if was $dw would be the tiddlywiki instance of the data wiki)
wiki is the active tiddler store of the code wiki
  (or tiddler store of data wiki if using $dw)
getTiddlerText is the tiddlywiki function to get the text of a tiddler
'REPL MOTD' is the tiddler to get the text from
40 is the terminal color code to colorize the text

Press up-arrow and change 'REPL-MOTD-MSG' to 'REPL-MOTD-MSG-1' then press enter.
