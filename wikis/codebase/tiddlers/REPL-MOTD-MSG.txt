
current: cmd.cog('REPL-MOTD-MSG')

The cmd.cog(title) function displays the text of the given 
tiddler title from the $cw code wiki. The text you are
reading is the text of tiddler 'REPL-MOTD-MSG' from the
code wiki.

cmd.cog(title) uses tiddlywiki functions to get the text.

The system has access to all tiddlywiki JavaScript functions
for four wikis; $ds, $dw, $cw, and $tw.
  $ds is the dash wiki http://localhost:3000 (dashboard)
  $cw is the code wiki http://localhost:3001 (codebase)
  $dw is the data wiki http://localhost:3002 (database)
  $tw is a tiddlywiki instance used by this REPL

Press up-arrow and change 'REPL-MOTD-MSG' to 'REPL-MOTD-MSG-1'
 then press enter to continue.
