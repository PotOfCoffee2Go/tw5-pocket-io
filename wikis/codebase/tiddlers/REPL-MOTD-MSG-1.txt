The majority of server-side code is stored in tiddlers on the $cw
 code wiki.

For example: cmd.cog('startup-commands') displays the server-side
code that provides short-cuts to common tiddlywiki functions for
use by the REPL. Notice the 'cog' function is defined there.

To see the code in the browser go to http://localhost:3001
  and select Project 'startup' tab 'commands'.
(to edit the code - press the little paperclip that displays on the tab)
